# UI/UX Online Shop

Project ini berfokus pada perancangan desain antarmuka (UI/UX) aplikasi online shop menggunakan Figma serta pembuatan prototype website.

## Deskripsi
Desain dibuat untuk mensimulasikan alur pengguna mulai dari halaman utama, pemilihan produk, hingga proses checkout.
