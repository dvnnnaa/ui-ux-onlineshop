# UI/UX Online Shop

## Project Overview
This project focuses on designing the User Interface and User Experience (UI/UX) of an online shop application. The design is created using Figma and presented as an interactive website prototype.

## Description
The design simulates the complete user flow, starting from the homepage, product browsing and selection, adding items to the cart, and continuing through the checkout process.  
The goal of this project is to create a clean, intuitive, and user-friendly shopping experience.
